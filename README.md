# 📖 NovelForge - AI Writing Studio

A full-featured novel writing suite with a built-in AI co-pilot. Write, plan characters, and brainstorm with AI—all in one place.

## 🚀 Live Demo
[Click here to try it live!](https://your-username.github.io/novelforge)

## ✨ Features
- **✍️ Markdown Editor** – Split-pane live preview with auto-save (IndexedDB).
- **🤖 AI Co-Pilot** – Brainstorm, rewrite, or continue scenes using OpenAI's API (GPT-3.5/4).
- **🌍 World Builder** – Manage characters, roles, and descriptions.
- **📖 Reader Mode** – Distraction-free reading with Kindle-like formatting.
- **📤 Export** – Export your entire novel as a single `.md` file.
- **💾 Auto-Save** – Every keystroke is saved to IndexedDB.
- **🎨 Dark/Light Theme** – Full theme toggle.

## 🛠️ Built With
- HTML5 / CSS3 / Vanilla JS
- IndexedDB (via a custom wrapper)
- Marked.js (Markdown rendering)
- OpenAI API (GPT-3.5-Turbo)
